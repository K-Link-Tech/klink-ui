## Usage

use-os detects user's os. Possible values are: `undetermined`, `macos`, `ios`, `windows`, `android`, `linux`.
If os cannot be identified, for example, during server side rendering `undetermined` will be returned.

<Demo data={HooksDemos.useOsDemo} />

## TypeScript

You can import `OS` union type from `@klink-ui/hooks`:

```tsx
import type { OS } from '@klink-ui/hooks';
// OS type is 'undetermined' | 'macos' | 'ios' | 'windows' | 'android' | 'linux'
```

## Definition

```tsx
function getOS(options?: {
  getValueInEffect: boolean;
}): 'undetermined' | 'macos' | 'ios' | 'windows' | 'android' | 'linux';
```

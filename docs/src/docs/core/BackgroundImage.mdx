## Usage

Use `BackgroundImage` component when you need to display the image behind any content.
Component sets background-image to given `src`, background-size to `cover` and background-position to `center`.
It can be used for cards, hero headers and similar components:

<Demo data={BackgroundImageDemos.usage} />

## Custom root element

`BackgroundImage` is a [polymorphic component](/guides/polymorphic/), you can change root element by setting `component` prop:

```tsx
import { BackgroundImage } from '@k-link/core';

function Demo() {
  return (
    <BackgroundImage
      component="a"
      href="https://k-link.dev"
      target="_blank"
      src="https://k-link.dev/image.png"
    />
  );
}
```

## Accessibility

`BackgroundImage` component sets `background-image`, it is ignored by screen readers,
if this is not a desired behavior, use [Image](/core/image/) component instead.

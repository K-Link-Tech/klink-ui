## Usage

Klink UI does not expose typography global styles.
Use `TypographyStylesProvider` to add Klink UI typography styles to your html content:

```tsx
import { TypographyStylesProvider } from '@klink-ui/core';

function Demo() {
  return (
    <TypographyStylesProvider>
      <div dangerouslySetInnerHTML={{ __html: '<p>Your html here</p>' }} />
    </TypographyStylesProvider>
  );
}
```

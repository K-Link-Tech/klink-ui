## Usage

Container is the most basic layout element, it centers content horizontally and adds horizontal padding from theme.
Component accepts these props:

- **size** – controls default max width
- **sizes** - configures container sizes
- **fluid** – overwrites size prop and sets max width to 100%

## Configure sizes

To configure container sizes use default props on [MantineProvider](/theming/k-link-provider/):

```tsx
import { MantineProvider, Container } from '@k-link/core';

function App() {
  return (
    <MantineProvider
      theme={{
        components: {
          Container: {
            defaultProps: {
              sizes: {
                xs: 540,
                sm: 720,
                md: 960,
                lg: 1140,
                xl: 1320,
              },
            },
          },
        },
      }}
    >
      <Container size="sm">Your app</Container>
    </MantineProvider>
  );
}
```
